## 🚀 DERNIÈRE ÉTAPE: DÉPLOYER SUR RAILWAY (2 MINUTES)

### Étape 1: Se Connecter à Railway

1. **Ouvrir Railway dans votre navigateur:**
   ```
   https://railway.app/new
   ```

2. **Se connecter:**
   - Cliquer sur **"Login with GitHub"**
   - Autoriser Railway à accéder à vos repositories
   - Vous serez redirigé vers le dashboard

### Étape 2: Déployer le Projet

1. **Créer un nouveau projet:**
   - Cliquer sur **"New Project"**
   - Sélectionner **"Deploy from GitHub repo"**

2. **Sélectionner le repository:**
   - Chercher: `passive-income-site`
   - Cliquer sur `gueyea66/passive-income-site`

3. **Railway déploie automatiquement!**
   - Vous verrez les logs en temps réel
   - Build: ~2 minutes
   - Déploiement: ~30 secondes
   - Attendez le message "Deployment successful"

### Étape 3: Obtenir l'URL

1. **Générer le domaine:**
   - Dans le dashboard Railway
   - Cliquer sur votre projet `passive-income-site`
   - Onglet **"Settings"**
   - Section **"Domains"**
   - Cliquer **"Generate Domain"**

2. **Copier l'URL:**
   - Format: `https://passive-income-site-production.up.railway.app`
   - Ou similaire
   - **C'est l'URL de votre site en ligne!**

3. **Tester le site:**
   - Ouvrir l'URL dans votre navigateur
   - Vous devriez voir votre site avec les 100 articles!

## 💰 APRÈS LE DÉPLOIEMENT: CONFIGURER ADSENSE

### Maintenant (5 minutes)

1. **Créer un compte AdSense:**
   - Aller sur: https://www.google.com/adsense
   - Cliquer "Get Started"
   - Se connecter avec votre compte Google
   - Entrer l'URL Railway de votre site
   - Soumettre pour approbation

2. **Vérifier le site:**
   - AdSense va demander de vérifier le code
   - Le code est DÉJÀ dans votre site (header.ejs)
   - Cliquer "Verify" dans AdSense
   - Attendre confirmation

### Plus Tard (1-2 semaines)

3. **Attendre l'approbation:**
   - Google examine votre site
   - Durée: 1-2 semaines
   - Vous recevrez un email

4. **Configurer le Publisher ID:**
   - Une fois approuvé, copier votre Publisher ID
   - Format: `ca-pub-XXXXXXXXXXXXXXXXX`
   - Aller dans Railway Dashboard
   - Votre projet → Variables
   - Ajouter: `ADSENSE_CLIENT_ID` = `ca-pub-VOTRE-ID`
   - Railway redéploie automatiquement
