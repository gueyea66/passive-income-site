## 💰 PROCHAINE ÉTAPE: ADSENSE

### Maintenant que votre site est en ligne:

1. **Aller sur Google AdSense:**
   ```
   https://www.google.com/adsense
   ```

2. **Cliquer "Get Started"**

3. **Entrer l'URL Railway de votre site**
   - Coller l'URL que vous avez copiée
   - Exemple: `https://passive-income-site-production.up.railway.app`

4. **Soumettre pour approbation**
   - Google va examiner votre site
   - Durée: 1-2 semaines
   - Vous recevrez un email

5. **Une fois approuvé:**
   - Copier votre Publisher ID (ca-pub-XXXXX)
   - Retourner sur Railway
   - Ajouter une variable: `ADSENSE_CLIENT_ID`
   - Railway redéploie automatiquement

## 🆘 SI PROBLÈME

### "Je ne vois pas mon repo dans Railway"
**Solution:**
- Attendre 1-2 minutes après le push GitHub
- Rafraîchir la page Railway (F5)
- Vérifier que vous êtes connecté avec le bon compte GitHub

### "Le déploiement échoue"
**Solution:**
- Regarder les logs Railway (onglet "Deployments")
- Vérifier qu'il n'y a pas d'erreur rouge
- Si erreur, me la montrer et je corrigerai

### "Le site ne charge pas"
**Solution:**
- Attendre 1-2 minutes après "Deployment successful"
- Vérifier que le statut est "Active" (vert)
- Essayer en navigation privée
- Vider le cache du navigateur

### "Je n'ai pas reçu l'email AdSense"
**Solution:**
- Vérifier les spams
- L'approbation prend 1-2 semaines
- Vérifier le statut dans AdSense Dashboard
